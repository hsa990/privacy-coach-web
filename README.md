# Personal Data Privacy Coach

A student-built ASP.NET Core MVC web application developed for CPIS-358.

## Features
- ASP.NET Identity authentication (Login/Register)
- Privacy awareness quiz (10 questions)
- Dashboard with score analysis and recommendations
- Session and cookie usage
- CRUD module for managing privacy tips
- Responsive UI using Bootstrap and custom CSS

## Technologies
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server LocalDB
- Bootstrap, JavaScript, CSS

## How to Run
1. Open the solution in Visual Studio Community
2. Restore NuGet packages
3. Run database migrations if needed
4. Press Run (IIS Express)
